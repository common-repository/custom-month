== Description ==

This is a blog date display plug-in, it can help you customize your blog's month display. The default month is in Chinese.
No more description, easy to use.

== Installation ==

1. Upload `custom-month.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
